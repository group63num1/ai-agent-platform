机器学习基础知识

机器学习（Machine Learning，ML）是人工智能的一个重要分支，通过算法使计算机能够从数据中学习和改进。

主要类型：

1. 监督学习（Supervised Learning）
   定义：使用标记的训练数据来学习从输入到输出的映射关系
   
   分类任务：
   - 逻辑回归
   - 支持向量机（SVM）
   - 决策树
   - 随机森林
   - 神经网络
   
   回归任务：
   - 线性回归
   - 多项式回归
   - 岭回归
   - LASSO回归

2. 无监督学习（Unsupervised Learning）
   定义：从未标记的数据中发现隐藏的模式和结构
   
   聚类：
   - K-means聚类
   - 层次聚类
   - DBSCAN
   
   降维：
   - 主成分分析（PCA）
   - t-SNE
   - 自编码器

3. 强化学习（Reinforcement Learning）
   定义：通过与环境交互来学习最优行为策略
   
   核心概念：
   - 状态（State）
   - 动作（Action）
   - 奖励（Reward）
   - 策略（Policy）
   - 价值函数（Value Function）
   
   算法：
   - Q-learning
   - 策略梯度
   - Actor-Critic
   - Deep Q-Network（DQN）

常用评估指标：
- 分类：准确率、精确率、召回率、F1分数
- 回归：均方误差（MSE）、平均绝对误差（MAE）
- 聚类：轮廓系数、调整兰德指数

数据预处理：
- 数据清洗
- 特征选择
- 特征工程
- 数据标准化
- 缺失值处理

过拟合和欠拟合：
- 过拟合：模型在训练数据上表现很好，但在测试数据上表现较差
- 欠拟合：模型无法很好地学习训练数据中的模式
- 解决方法：正则化、交叉验证、早停等
