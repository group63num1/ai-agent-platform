# 一、查询已启用知识库名称列表

### **GET `/api/knowledge-bases/getlist`**

返回所有 **已启用（enabled=1）** 的知识库名称，按 **更新时间倒序** 排序。

------

### **后端改动说明**

- **KnowledgeBaseMapper / XML**
  - 新增 `selectEnabledNames()` 查询 `enabled=1` 的知识库名称。
- **KnowledgeBaseService & 实现类**
  - 新增 `listEnabledNames()` 方法。
- **KnowledgeBaseController**
  - 新增 `/getlist` 路由，返回 `ApiResponse<List>`。

------

### **请求示例**

```
GET http://localhost:28080/api/knowledge-bases/getlist
```

------

### ** 返回示例**

```
{
  "code": 200,
  "msg": "ok",
  "data": [
    "示例知识库A",
    "示例知识库B"
  ]
}
```

------

# 二、修改会话名称

### **PUT `/api/agents/{id}/sessions/{sessionId}`**

用于修改某个智能体会话的名称。

------

### **请求示例**

```
PUT http://localhost:28080/api/agents/{id}/sessions/{sessionId}
Content-Type: application/json
```

#### 请求 Body：

```
{
  "name": "新的会话名称"
}
```

------

### ** 返回示例**

```
{
  "code": 200,
  "msg": "ok",
  "data": {
    "sessionId": "...",
    "agentId": "...",
    "name": "新的会话名称",
    "createdAt": "...",
    "updatedAt": "..."
  }
}
```
